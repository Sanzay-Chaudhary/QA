# QA Automation Project

## Overview
This project demonstrates automation testing using Selenium and PyTest.

## Features Covered
- Login functionality testing
- Positive and negative test scenarios
- Page Object Model (POM)
- PyTest framework with fixtures

## Tools & Technologies
- Python
- Selenium
- PyTest

## Test Cases
- Valid login test
- Invalid login test

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   pytest
